Features

  • Bundling using dependency graph & IIFE
  • Create dependencies from js (default), css, html and others
  • Transpilation / Transciprion: Babel, TypeSctipt, Sass
  • Linter, CSSComb
  • Livereload & Hot Module Replacement
  • Optimization:
    • UglifyJS
    • Tree Shaking
    • Code Spliting:
      • Cache
      • Lazy Load
